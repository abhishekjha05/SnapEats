# SnapEats - Food Delivery App 🍕

A modern, fast food delivery app inspired by Zomato with a beautiful **Manjusa art theme** UI. Manjusa art is a traditional Indian art form known for its vibrant colors, intricate patterns, and cultural aesthetics.

## Features

✨ **Key Features:**
- Real-time order tracking
- Restaurant discovery with Manjusa art-themed cards
- Seamless payment integration
- User profiles and order history
- Live delivery tracking
- Ratings and reviews
- Manjusa art-inspired color palette and patterns

## Tech Stack

- **Frontend:** React Native / React.js
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Payment:** Stripe/Razorpay integration
- **Maps:** Google Maps API
- **Real-time:** Socket.io for live tracking

## Manjusa Art Theme

The app incorporates traditional Manjusa art elements:
- Vibrant color palette: Warm oranges, deep reds, golden yellows, emerald greens
- Intricate pattern borders and dividers
- Hand-drawn style icons and illustrations
- Cultural motifs and ornamental designs
